# dlh-higher_level_programming

Higher-level programming projects completed as part of the **AI Academy at Digital Learning Hub Luxembourg**, following the Holberton School ML Engineering curriculum.

## Overview

This repository covers core Python programming concepts through hands-on exercises and projects, building the foundation needed for data science and machine learning work.

## Topics Covered

- **Data Structures** — Lists, tuples, dictionaries, and sets with real-world use cases
- **Control Flow** — If/else logic, loops, and iteration patterns
- **Functions** — Defining, calling, and composing reusable functions
- **Object-Oriented Programming** — Classes, inheritance, and encapsulation
- **File I/O** — Reading from and writing to files
- **Error Handling** — Try/except blocks and exception management
- **Modules & Packages** — Importing and organizing code

## Tools & Environment

- **Language:** Python 3
- **Linting:** pycodestyle / pylint
- **Editor:** VS Code
- **Version Control:** Git & GitHub

## Structure

Each subdirectory corresponds to a specific topic or exercise set from the curriculum.
